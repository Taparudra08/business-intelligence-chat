# 📊 AI Data Analyzer

Upload data → Ask questions → Get charts using AI.

## 🚀 Features
- Upload CSV / Excel / PDF
- Ask questions in natural language
- Auto-generate charts
- AI-powered insights

## 🛠️ Tech
- Flask (Backend)
- React (Frontend)
- SQLite (Database)
- Gemini API (AI)
- Plotly (Charts)

## ▶️ Run

### Backend
pip install flask pandas requests plotly pdfplumber flask-cors
python app.py

### Frontend
npm install
npm start

## 📌 Example Query
"total sales by product"
